# Как пользоваться Doba каждый день (без Xcode)

Когда жмёшь ▶ в Xcode — приложение работает только пока Xcode его «держит».
Чтобы пользоваться им как обычной программой, вынеси его в «Программы» **один раз**:

## Вынести приложение в «Программы»

1. Запусти приложение из Xcode (▶) хотя бы раз.
2. Слева в Xcode внизу найди синюю папку **Products** → разверни → **Doba.app**.
3. Правый клик по **Doba.app → Show in Finder**.
4. В Finder перетащи **Doba.app** в папку **Программы (Applications)**.
5. Можно закрыть Xcode. Дальше запускай **Doba** через Spotlight (⌘+пробел → «Doba»)
   или из «Программ» — как любое приложение.

> Если macOS при первом запуске ругается («не удаётся проверить разработчика») —
> правый клик по Doba → **Открыть** → **Открыть**. Один раз.

> Запускай Doba **только из «Программ»** (Spotlight), а не из Xcode через ▶.
> Если запустить из Xcode, в Launchpad снова появляется второй значок (это
> сборочная копия). Для повседневного использования — только /Программы.

## Чтобы запускалось само при входе

**Системные настройки → Основные → Объекты входа (Login Items)** → «+» → добавь **Doba**.
Теперь иконка-чеклист в меню-баре будет появляться сама после перезагрузки.

## Когда появятся новые фичи (Phase 4, 5 и правки)

Код меняется → нужно пересобрать:

1. В Xcode нажми ▶ один раз (соберёт новую версию).
2. Повтори «Вынести в Программы» (перетащи новый Doba.app, заменив старый).

Твои данные при этом не теряются — задачи лежат в отдельном JSON-файле, не внутри
приложения.

## Что уже умеет (Phase 0–3)

- Задачи на сегодня: **таймлайн** (со временем) + **плавающий пул** (без времени).
- Быстрый ввод (Enter) + детальный редактор (карандаш): проект, часы, время, billable.
- **Проекты** с цветом.
- **Перенос** несделанного на сегодня.
- **Календарь** (read-only): сегодняшние миты на таймлайне.
- **Таймер**: старт/стоп на задаче, факт-часы, ручная правка; свёртка **план/факт** и
  **billable/overhead**. Когда таймер доходит до оценки задачи, он **сам
  останавливается**: звук, системное уведомление и окно поверх всех приложений на
  том экране, где ты работаешь (видно даже во время созвона).

Виджет на рабочем столе пока **только показывает** план (галочки в нём — это Phase 5).
